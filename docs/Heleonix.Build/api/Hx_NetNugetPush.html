Table of Contents

Hx_NetNugetPush

Properties

Hx_NetNugetPush_ArtifactsDir

A path to the NugetDeploy artifacts directory.

Hx_NetNugetPush_APIKey

The API key.

Hx_NetNugetPush_SourceURL

The source path to publish to (nuget.org etc.).

Hx_NetNugetPush_Version

The semantic version. It is passed as /p:Version property to the Pack target. Default is a version retrieved from the $Hx_ChangeLog_ArtifactsDir/semver.txt.

Hx_NetNugetPush_ReleaseNotesFile

A text file with package release notes. It is passed as /p:PackageReleaseNotes property into the Pack target. Default is $Hx_ChangeLog_ArtifactsDir/ReleaseNotes.txt.

Items

Hx_NetNugetPush_PackageFiles

The package files to publish. If not specified, the .nupkg and .snupkg files are searched in the 'bin' folders. If files are not found, the target tries to find a .sln file and run the 'Pack' MSBuild target.
